ปฏิเสธไม่ได้ว่า AI Coding Tools เข้ามามีบทบาทในการพัฒนาซอฟต์แวร์เป็นอย่างมาก เป็นทั้งคู่คิด ผู้ช่วยในการ implement code หรือช่วย generate test cases ยาวๆ เยอะๆ ให้เรา เพิ่ม productivity และลดเวลาทำงานไปได้เยอะ

แต่ทุกคนที่ใช้ AI Coding Tools จะต้องเคยเจอเหตุการที่ AI มันหลอน (Hallucinate) เจนโค้ดผิดๆ ถูกๆ หรือดันเจนโค้ดที่มีช่องโหว่ที่ outdated ไปนานมากๆ แล้ว แต่จะให้มาคอยบอกมันทุกครั้งว่า เจนโค้ดตาม convention นี้นะ ให้เขียนโค้ด style นี้นะ ก็ดูจะเป็นอะไรที่เหนื่อยเกินไป วันนี้เราจะมาลองใช้
